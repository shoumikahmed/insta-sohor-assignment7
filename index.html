1. The difference between local storage and session storage:


Local Storage:
  1. Local storage means client side store data by cookies.
  2. It stores data with no expiration date.
  3. The data of local storage is not changeable by going another website.
  4. If you refresh the the website local storage data will be remained until you delete the data using javascript.
Session Storage:
  1. Session storage means store data by server side.
  2. It distroys data when the user close the window.
  3. The data of session storage is changeable by going another website.
  4. If you refresh the the website session storage data will be removed
